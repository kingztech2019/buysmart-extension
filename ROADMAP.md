# BuySmart Roadmap

This document outlines the planned features and improvements for BuySmart.

## Current Version: 1.0.0

### ✅ Completed Features
- Chrome Extension with auto-detection
- Support for Amazon, eBay, AliExpress, Walmart, Jumia
- AI-powered product analysis via OpenRouter
- Multi-source price comparison via SearXNG
- Tavily API integration for price search
- Deepcrawl integration for enhanced scraping
- Docker Compose deployment
- Manual search mode
